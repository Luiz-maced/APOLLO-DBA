🚀 APOLLO-DBA
APOLLO-DBA is a web-based application designed to assist Database Administrators (DBAs) in managing and monitoring database systems efficiently. Built with Flask, it offers a user-friendly interface for various database operations.

🛠️ Features
Intuitive dashboard for database monitoring

User authentication and authorization

Real-time alerts and notifications

Responsive design compatible with multiple devices
